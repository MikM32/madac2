# madac2
# Interprete del pseudocodigo NASPI
Creado con la finalidad de facilitar las pruebas de algoritmos escritos en las materias de fundamentos de algoritmos y algoritmos I.

Caracteristicas:

- Soporte para bucles (para, mientras)
- declaración y asignación de variables.
- Maquina Virtual con recolector de basura basado en algoritmo Mark-And-Sweep
